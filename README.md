# Hi, I'm Ömer Yıldız

**Software Engineer & Technical Team Lead** focused on cryptography, hardware security modules, secure embedded systems, and applied software architecture.

I design and develop security-oriented software across the stack—from embedded Linux and smart cards to PKCS#11 integrations, cryptographic services, NFC applications, and distributed systems.

## Areas of Expertise

- **Cryptography and HSMs:** PKCS#11, key management, digital signatures, secure key storage
- **Smart cards and NFC:** Java Card, NDEF, APDU, GlobalPlatform, Ed25519
- **Secure embedded systems:** Embedded Linux, ARM-based systems, hardware-backed security
- **Systems programming:** C, C++, Linux, inter-process communication, RPC
- **Backend and integration:** Java, Python, RESTful services, Apache Thrift, data interchange formats
- **Software architecture:** Object-oriented and procedural design, technical leadership, cross-team coordination
